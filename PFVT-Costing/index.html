About This Tool

The IBM MAS PFVT Cost & Sizing Advisor helps you estimate costs for IBM Maximo Application Suite (MAS) deployments on Red Hat OpenShift Kubernetes Service (ROKS) clusters.

How to Use

  1. Select MAS Configuration: Choose from 6 baseline configurations based on your MAS components
  2. Choose Deployment Options: Select number of zones (1-3) and worker node flavor
  3. Calculate Costs: View detailed cost breakdown and PFVT Cost Score
  4. Review Risks & Recommendations: Address any issues before deployment
  5. Save Scenario: Save for future reference and comparison

PFVT Cost Score

The cost score provides instant visual feedback on your configuration costs:

  • üü¢ GREEN (Optimal): Cost is within optimal range
  • üü° AMBER (Moderate): Cost is moderate, review for optimization
  • üî¥ RED (High): Cost is high, optimization recommended

Thresholds adjust based on number of zones selected.

Multi-Zone Deployments

Worker node costs are multiplied by the number of zones:

Total Cost = Workers √ó Zones √ó Hourly Rate √ó Hours

Example: 3 workers √ó 2 zones √ó $0.94/hr √ó 730 hrs = $4,117.20/month
